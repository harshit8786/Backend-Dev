1. Advantages of Mongoose over MongoDB driver
Mongoose makes MongoDB easier to use. It lets us define a schema for data, so structure stays clear. It has built-in validation to avoid wrong data. It supports relationships using populate. Queries are simpler and more readable. It also provides middleware and automatic type conversion.

2. Difference between findOneAndUpdate() and updateOne()
findOneAndUpdate() finds a document and updates it, and can also return the updated data.
updateOne() only updates the document and returns status, not the updated data.

3. Purpose of middleware in Mongoose
Middleware runs functions before or after operations like save or update. It is used for tasks like validation, password hashing, and logging.

4. Pagination in Mongoose
Pagination is done using skip() and limit() to fetch data in parts (pages).

5. Embedding vs Referencing
Use Embedding when data is small and always used together.
Use Referencing when data is large, complex, or used separately.
